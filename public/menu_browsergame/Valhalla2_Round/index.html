■　説明

●　ヴァルハラ2とは？
「じゃんけん」＋「ポーカー」みたいなカードゲームです。
前作より属性が増えて難しくなりましたが、属性ボーナスが起こる確率も増しました。
前作を未プレイの方は、まずは前作「ヴァルハラ」からプレイし、それに慣れたらヴァルハラ2に挑戦してみて下さい。

前作との違いは、カードを引く回数が1回増えたことと、上位属性のカード「炎1」、「雷1」、「聖1」、「死1」、「氷1」の5種類が追加されていることです。
「炎」は「火」の、「雷」は「風」の、「聖」は「光」の、「死」は「闇」の、「氷」は「水」の上位属性です。

上位属性は、下位属性の性質＋下位属性に強い性質を持っています。
例えば、「炎」なら「風に強く、水に弱い」（火属性の性質）＋「火に強い」性質を持っています。

上位属性は、上位属性カード＋下位属性カード2枚以上で発動することができます。
例えば、炎1枚＋火2枚で発動すると、炎属性になります。

●　操作方法
全ての操作をマウスで行って下さい。
一番手前のカード（手札）の中から1枚を選んでクリックするだけです。

●　ゲームの流れ
初期ドローとして山札から5枚引く。
　　　↓
「1枚引く　→　両者同時に1枚捨てる」を6回繰り返す。
　　　↓
発動する属性を選択し、両者同時に発動する。
発動するには3枚以上の同属性カードが必要。
　　　↓
勝利判定。
発動した属性の総攻撃力に属性ボーナス（弱点属性を突けば+4）を足します。
両者の値を比較し、攻撃力が高い方がそのラウンドの勝者となります。
ゲーム開始前に設定した「必要勝利数」を先取すればこのゲームの勝者に！

●　カードの種類
火属性カード　…　風に強く、水に弱い。　攻撃力　1～3。
炎属性カード　…　火と風に強く、水に弱い。　攻撃力　1。　火属性カード2枚以上と合わせて発動すると炎属性になる。
風属性カード　…　光に強く、火に弱い。　攻撃力　1～3。
雷属性カード　…　風と光に強く、火に弱い。　攻撃力　1。　風属性カード2枚以上と合わせて発動すると雷属性になる。
光属性カード　…　闇に強く、風に弱い。　攻撃力　1～3。
聖属性カード　…　光と闇に強く、風に弱い。　攻撃力　1。　光属性カード2枚以上と合わせて発動すると聖属性になる。
闇属性カード　…　水に強く、光に弱い。　攻撃力　1～3。
死属性カード　…　闇と水に強く、光に弱い。　攻撃力　1。　闇属性カード2枚以上と合わせて発動すると死属性になる。
水属性カード　…　火に強く、闇に弱い。　攻撃力　1～3。
氷属性カード　…　水と火に強く、闇に弱い。　攻撃力　1。　水属性カード2枚以上と合わせて発動すると氷属性になる。
変化カード　…　発動する属性と同じになる。「火」2枚、「変」1枚でも発動可能。攻撃力　0、1。
